# Low-Cost Multispectral Solvent Waste Sensor

## Overview

This project documents the development of a low-cost multispectral optical sensor system for monitoring solvent waste in teaching laboratories.

The system uses a Raspberry Pi Pico WH, TSL2591 light sensor, and LEDs at selected wavelengths to collect optical readings from samples.

This project is part of my MSc Analytical Chemistry research project at Kingston University.

## Current Project Stage

At the current stage, the prototype has focused on:

- Raspberry Pi Pico WH setup
- MicroPython installation
- Single LED circuit testing
- LED switching using a 2N2222A transistor
- TSL2591 sensor connection
- Ambient light baseline readings
- CSV data logging
- Planning a four-wavelength LED system

## Hardware Used

- Raspberry Pi Pico WH
- TSL2591 light sensor
- 465 nm LED
- 625 nm LED
- 850 nm NIR LED
- 940 nm NIR LED
- 2N2222A transistor
- Breadboard
- Resistors
- Jumper wires
- USB cable
